# advpublico-site

Landing page do Pack Completo AdvPúblico (38 Agentes de IA para a Advocacia Pública), publicada via GitHub Pages.

- `index.html` — página de vendas (link Hotmart trocado na constante `HOTMART_URL` no fim do arquivo)
- `assets/` — logo e avatar da marca

Quando o domínio advpublico.com.br for registrado: adicionar arquivo `CNAME` com o domínio + apontar DNS no registro.br.
